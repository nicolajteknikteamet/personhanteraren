Personhanterare is a system for handling persons for Soundforce, an event company. The requirement are:
Calender view with an easy way of showing the current week (mon-sun). 
Theres 22 persons in the company.
They should be able to choose their location for a given day. The calender should show where all persons are currently. 

Code is based on next.js with nextauth for signins.
